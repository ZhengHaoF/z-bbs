# Z-BBS

#### 介绍
Z-BBS是一个超轻量级的论坛系统，前端采用Bootstrap4，后端采用ThinkPHP，全部操作接口化，对接简单方便，前端自适应手机平板等 

### 详情

#### 在线测试

[Z-Blog (zhfblog.top)](https://bbs.zhfblog.top/)
